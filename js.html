Les opérateurs

Dans une affectation de valeur à une variable, on trouve une expression à droite du signe égal. Cette expression est un ensemble de valeurs, reliées par des opérateurs et équivalant à une seule valeur.
Différents types d'opérateurs :

  • opérateurs numériques : + - * / ^ ()
  • opérateurs alphanumériques : + (concaténation)
  • opérateurs logiques : && (et) || (ou)

Les fonctions

Une fonction JavaScript est un ensemble d'instructions effectuant une tâche ou calculant une valeur. Afin d'utiliser une fonction, il est nécessaire de l'avoir auparavant définie au sein de la portée dans laquelle on souhaite l'appeler. On définit une fonction avec le mot clé function suivi par le nom de la fonction, une liste d'arguments à passer à la fonction (entre parenthèses et séparés par des virgules) et les instructions JavaScript définissant la fontion (entre accolades).
