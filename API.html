Le API

Présentation

Le terme API signifie Application Programming Interface. Il s'agit d'une interface qui permet à plusieurs applications de communiquer entre elles. L'API d'une application permet un accès aux données de celle-ci. L'accès aux données se fait de façon structuré. Une application cliente va utilisée l'API d'une autre application afin d'accéder directement aux données sans changer d'application. Voici une courte vidéo explicative.
